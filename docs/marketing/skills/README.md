# Marketing skills (Cursor)

The marketing voice skill lives in `.cursor/` locally (gitignored). Copy into your Cursor skills folder:

```bash
mkdir -p .cursor/skills/mdpdev-marketing-voice
cp docs/marketing/skills/mdpdev-marketing-voice/* .cursor/skills/mdpdev-marketing-voice/
```

Or symlink:

```bash
mkdir -p .cursor/skills
ln -sf ../../docs/marketing/skills/mdpdev-marketing-voice .cursor/skills/mdpdev-marketing-voice
```
